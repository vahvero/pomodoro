# Pomodoro

Pomodoro timer

Windows:

Create virtual environment

    py -m venv ENV
    .\ENV\Scripts\activate

Install requirements

    pip install -r requirements.txt

Run

    py main.py 25 5


Unix:

Create virtual environment

    python3.9 -m venv ENV
    source ENV/bin/activate

Install requirements

    pip install -r requirements.txt

Run

    python main.py 25 5


Exit program by `ctrl+c`.
